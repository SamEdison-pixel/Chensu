# 辰宿 - 单身经济背景下心理健康社群平台

## 📋 项目简介

"辰宿"是一个专注于为单身人群提供心理健康支持和社交陪伴的综合性平台。在单身经济蓬勃发展的时代背景下，我们致力于构建一个温暖且有归属感的生活生态系统，让单身生活更健康、温暖且有尊严。

## ✨ 核心功能

### 1. 情感共鸣社区
- 匿名树洞与AI情绪预警
- "孤勇者故事集"专栏
- 社区守护者机制

### 2. 线下活动社群
- 轻量化兴趣匹配与低门槛活动设计
- 线上破冰场景设计与线下"轻压力"模式
- 社交信用分体系

### 3. 专业课程与咨询
- 阶梯式支持体系（AI树洞助手、9.9元/周倾听计划）
- 5分钟正念胶囊课程、独居生存课等
- 匿名"情绪盲盒"功能

### 4. 单身经济板块
- 消除单身消费场景中的隐形歧视
- 反向众筹与认证审核机制
- 积分体系与独居福利日

## 🛠️ 技术栈

- **前端框架**: React 18+
- **编程语言**: TypeScript
- **样式框架**: Tailwind CSS
- **状态管理**: React Context API
- **路由**: React Router DOM
- **动画**: Framer Motion
- **图表**: Recharts
- **图标**: Lucide React, Font Awesome
- **构建工具**: Vite
- **表单验证**: Zod
- **通知提示**: Sonner

## 🚀 快速开始

### 前提条件

确保您的环境中已安装以下工具：
- Node.js (v16+)
- pnpm (推荐) 或 npm/yarn

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

项目将在 http://localhost:3000 启动，您可以在浏览器中访问查看效果。

### 构建生产版本

```bash
pnpm build
```

构建后的文件将位于 `dist` 目录中。
